### 2. agent-control

![agent-control 的 GitHub 仓库预览](https://opengraph.githubassets.com/1/serifcolakel/agent-control)

*图：agent-control 的公开 GitHub 仓库预览；图片由 GitHub 为该仓库生成。原始来源：[serifcolakel/agent-control](https://github.com/serifcolakel/agent-control)。*

- **首次公开：**GitHub 仓库创建于 **2026-07-18 14:51 UTC**。
- **项目：**[serifcolakel/agent-control](https://github.com/serifcolakel/agent-control)（MIT License）
- **项目自述：**为 Claude Code、Codex、OpenCode 等 CLI Agent 提供任务路由、记忆共享与人工审批，并通过单一面板编排。
- **为什么值得关注：**多 Agent 编程从“并行跑几个工具”走向可追溯的任务队列与审批链。这个项目还很新、社区信号有限，适合作为方向性观察，而非已验证的生产方案。

### 3. anti-slop

- **首次公开：**GitHub 仓库创建于 **2026-07-18 20:40 UTC**。
- **项目：**[iCodeCraft/anti-slop](https://github.com/iCodeCraft/anti-slop)（MIT License）
- **项目自述：**面向 AI 编程 Agent 的可插拔 skills，覆盖质量拦截、安全审查与 PR hygiene。
- **为什么值得关注：**把“不要产出低质量代码”转化为可复用的流程检查，是 Agent 工程化中务实的一步。其效果仍需要独立评估；本期仅确认该仓库的创建时间与作者公开描述。
